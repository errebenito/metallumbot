## ADR-001: Programming Language

**Status:** accepted  
**Date:** 12/04/2026  

### Context
We want to develop the bot in a language with good tooling and support that offers a mature ecosystem. In particular, the language should support what is required to develop the functionality of the bot (i.e. network calls, etc.), be easy to write and read, and have good tools for testing code written in it.

### Considered options
Most high-level languages fulfill these needs: C#, Java, Python, and many others.

### Decision outcome
Chosen option: Java, because

- Widely supported, with no shortage of IDEs, libraries, and other tools and components.
- Familiar syntax.
- Good tooling for testing.
