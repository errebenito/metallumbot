## ADR-006: Version Control

**Status:** accepted  
**Date:** 12/04/2026  

### Context
We want to use a version control system to keep track of changes to the code and ensure we can safely revert to a good state in case of bugs/regressions.

### Considered options
Git vs. Subversion vs. CVS. vs. Mercurial.

### Decision outcome
Chosen option: Git due to its maturity, wide support, lightweight branching system, and independence of a centralised server due to its distributed nature.
